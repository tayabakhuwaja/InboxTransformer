# InboxTransformer
InboxTransformer is an AI-powered Gmail assistant that fetches, classifies, and summarizes emails, then generates concise, professional auto-replies using Transformer models, leveraging the 'Attention is All You Need' architecture for intelligent natural language understanding.

## How It Works

InboxTransformer uses **Transformer models**, based on the revolutionary architecture from *"Attention is All You Need"*, to intelligently handle your emails.  

- **Smart Classification**: Automatically categorizes emails into Finance, Security, Promotions, Meetings, Updates, or General using zero-shot learning.  
- **Concise Summarization**: Summarizes lengthy email content for quick understanding.  
- **Context-Aware Auto-Replies**: Generates short, professional replies using GPT-2, tailored to the emailâ€™s content and category.  

By combining these capabilities, InboxTransformer transforms your inbox into a more organized, responsive, and manageable space, saving time and effort in daily email management.
